= Qhttpd
:experimental:

:x_download_url: https://raw.githubusercontent.com/quinngrier/qhttpd/main/qhttpd
:x_man_page_url: https://github.com/quinngrier/qhttpd/blob/main/qhttpd.1.adoc

Qhttpd is a simple web server.

Here's how to get started:

. Make sure you have the following tools installed:

.. Bash version 5.0 or later.
You can check this by running `bash --version`.

.. OpenSSL version 3.0.13 or later.
You can check this by running `openssl version`.

.. Python version 3.12.3 or later.
You can check this by running `python3 --version`.

. Download the link:{x_download_url}[`qhttpd`] script and optionally put
it in your `PATH`.

. Run `qhttpd` to serve the current directory over HTTPS on 0.0.0.0:8443
with a self-signed certificate.
